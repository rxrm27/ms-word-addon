# Patent Reference Checker — Word Add-in Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build an Office.js Word add-in that detects patent reference number conflicts (same number → different component name, or same component name → different number) and inserts a sorted reference table on demand.

**Architecture:** Vanilla JS Office Add-in (no build tools, no npm packages) — a manifest.xml + static HTML/CSS/JS served from a local Python HTTP server. Core logic lives in a pure-JS module (`numbering-engine.js`) testable in any browser without Word. Office.js integration in `taskpane.js` reads document text, calls the engine, and uses Word's search + comment APIs to mark conflicts.

**Tech Stack:** Office.js (CDN), Vanilla JS ES5/ES6, Python 3 (built-in `http.server` for local serving), HTML/CSS

```python
#!/usr/bin/env python3
"""Generates minimal solid-color PNG icons. No Pillow required — pure stdlib."""
import struct, zlib, os

def make_png(w, h, r, g, b):
    def chunk(tag, data):
        c = struct.pack('>I', len(data)) + tag + data
        return c + struct.pack('>I', zlib.crc32(tag + data) & 0xffffffff)
    ihdr = struct.pack('>IIBBBBB', w, h, 8, 2, 0, 0, 0)
    raw = b''.join(b'\x00' + bytes([r, g, b] * w) for _ in range(h))
    return (b'\x89PNG\r\n\x1a\n'
            + chunk(b'IHDR', ihdr)
            + chunk(b'IDAT', zlib.compress(raw))
            + chunk(b'IEND', b''))

out = os.path.join(os.path.dirname(os.path.abspath(__file__)), 'assets')
os.makedirs(out, exist_ok=True)
for size, name in [(32, 'icon-32.png'), (80, 'icon-80.png')]:
    with open(os.path.join(out, name), 'wb') as f:
        f.write(make_png(size, size, 26, 115, 232))  # #1a73e8 blue
    print(f"Created assets/{name}")
```

```python
#!/usr/bin/env python3
"""Local HTTP server for the Patent Reference Checker add-in.
Run: python serve.py
Keep it running while using the add-in in Word.
"""
import http.server, os, sys

PORT = 3000
ROOT = os.path.dirname(os.path.abspath(__file__))

class Handler(http.server.SimpleHTTPRequestHandler):
    def __init__(self, *args, **kwargs):
        super().__init__(*args, directory=ROOT, **kwargs)
    def end_headers(self):
        self.send_header('Access-Control-Allow-Origin', '*')
        super().end_headers()
    def log_message(self, fmt, *args):
        print(f"  {self.address_string()} {fmt % args}")

if __name__ == '__main__':
    with http.server.HTTPServer(('localhost', PORT), Handler) as s:
        print(f"Patent Reference Checker running at http://localhost:{PORT}")
        print("Keep this terminal open while using the add-in in Word.")
        print("Press Ctrl+C to stop.\n")
        try:
            s.serve_forever()
        except KeyboardInterrupt:
            print("\nStopped.")
```

```javascript
// src/numbering-engine.js
// Patent reference number conflict detector — pure JS, no dependencies.
(function (global) {
  'use strict';

  // Patterns to replace with spaces BEFORE regex matching.
  // Order matters: FIG/Figure refs first, then patent nos, then dates.
  var SKIP = [
    /\bFIG\.?\s*[\d(][^\s,;.]*/gi,           // FIG. 1, FIG.2(A), FIG.17(B)
    /\bFigure\s+[\d(][^\s,;.]*/gi,           // Figure 1, Figure 10
    /\bNo\.\s+[\d][^,;\s]*/gi,               // No. 18/413,382
    /\b\d{1,2}[\/\-]\d{1,2}[\/\-]\d{2,4}\b/g, // dates 01/16/2024
    /\b(?:January|February|March|April|May|June|July|August|September|October|November|December)\s+\d{1,2},?\s+\d{4}\b/gi
  ];

  // Capture: up to 6 words (letters/hyphens) then a 3-4 digit ref number (with optional sub-part).
  // Negative lookahead prevents matching numbers immediately followed by / . , or another digit.
  var REF_PAT = /\b([A-Za-z][A-Za-z\-]*(?:\s+[A-Za-z][A-Za-z\-]*){0,5})\s+(\d{3,4}(?:-\d+)?)\b(?![\/.,\d])/g;

  function cleanText(text) {
    var s = text;
    for (var i = 0; i < SKIP.length; i++) s = s.replace(SKIP[i], ' ');
    return s;
  }

  function normalizePhrase(raw) {
    return raw
      .toLowerCase()
      .replace(/^\s*(?:a|an|the)\s+/, '')
      .replace(/\s+/g, ' ')
      .trim();
  }

  // Returns array of { phrase, number, rawMatch, position }
  function parse(text) {
    var cleaned = cleanText(text);
    var entries = [];
    var m;
    REF_PAT.lastIndex = 0;
    while ((m = REF_PAT.exec(cleaned)) !== null) {
      var phrase = normalizePhrase(m[1]);
      if (!phrase) continue;
      entries.push({ phrase: phrase, number: m[2], rawMatch: m[0], position: m.index });
    }
    return entries;
  }

  // Returns { numberToPhrase: { "102": ["interface","module",...] },
  //           phraseToNumber: { "interface": ["102","202",...] } }
  function buildDictionary(entries) {
    var n2p = {}, p2n = {};
    entries.forEach(function (e) {
      if (!n2p[e.number]) n2p[e.number] = [];
      if (n2p[e.number].indexOf(e.phrase) === -1) n2p[e.number].push(e.phrase);
      if (!p2n[e.phrase]) p2n[e.phrase] = [];
      if (p2n[e.phrase].indexOf(e.number) === -1) p2n[e.phrase].push(e.number);
    });
    return { numberToPhrase: n2p, phraseToNumber: p2n };
  }

  // Returns array of conflict objects.
  function detectConflicts(dictionary) {
    var n2p = dictionary.numberToPhrase, p2n = dictionary.phraseToNumber;
    var conflicts = [], seen = {};

    Object.keys(n2p).forEach(function (num) {
      if (n2p[num].length > 1 && !seen['num:' + num]) {
        seen['num:' + num] = true;
        conflicts.push({
          type: 'number_reuse',
          number: num,
          phrases: n2p[num].slice(),
          description: 'Number ' + num + ' used for: ' + n2p[num].join(' AND ')
        });
      }
    });

    Object.keys(p2n).forEach(function (phrase) {
      if (p2n[phrase].length > 1 && !seen['ph:' + phrase]) {
        seen['ph:' + phrase] = true;
        conflicts.push({
          type: 'phrase_reuse',
          phrase: phrase,
          numbers: p2n[phrase].slice(),
          description: '"' + phrase + '" numbered as: ' + p2n[phrase].join(' AND ')
        });
      }
    });

    return conflicts;
  }

  // Returns array of { number, phrase } sorted numerically by number.
  function getReferenceDictionary(dictionary) {
    var n2p = dictionary.numberToPhrase;
    var rows = Object.keys(n2p).map(function (num) {
      return { number: num, phrase: n2p[num][0] };
    });
    rows.sort(function (a, b) {
      var an = parseInt(a.number.split('-')[0], 10);
      var bn = parseInt(b.number.split('-')[0], 10);
      return an !== bn ? an - bn : a.number.localeCompare(b.number);
    });
    return rows;
  }

  // Main entry point.
  function analyze(text) {
    var entries = parse(text);
    var dictionary = buildDictionary(entries);
    var conflicts = detectConflicts(dictionary);
    var referenceTable = getReferenceDictionary(dictionary);
    return { entries: entries, dictionary: dictionary, conflicts: conflicts, referenceTable: referenceTable };
  }

  // UMD export: works in browser (sets window.NumberingEngine) and Node.js.
  var exports = { parse: parse, buildDictionary: buildDictionary, detectConflicts: detectConflicts, getReferenceDictionary: getReferenceDictionary, analyze: analyze };
  if (typeof module !== 'undefined' && module.exports) {
    module.exports = exports;
  } else {
    global.NumberingEngine = exports;
  }
}(typeof window !== 'undefined' ? window : global));
```

```javascript
// ── Scan ─────────────────────────────────────────────────────────────────────

function scanDocument() {
  setStatus('Scanning document...');
  Word.run(function (ctx) {
    var body = ctx.document.body;
    body.load('text');
    return ctx.sync().then(function () {
      scanResult = NumberingEngine.analyze(body.text);
      renderResults(scanResult);
      var msg = scanResult.conflicts.length === 0
        ? 'No conflicts found. ' + scanResult.referenceTable.length + ' references detected.'
        : scanResult.conflicts.length + ' conflict(s) found. ' + scanResult.referenceTable.length + ' total references.';
      setStatus(msg, scanResult.conflicts.length === 0 ? 'success' : 'error');
      document.getElementById('btn-mark').disabled = scanResult.conflicts.length === 0;
      document.getElementById('btn-table').disabled = scanResult.referenceTable.length === 0;
    });
  }).catch(function (err) {
    setStatus('Error scanning document: ' + err.message, 'error');
  });
}

// ── Render results ───────────────────────────────────────────────────────────

function renderResults(result) {
  renderConflicts(result.conflicts);
```

```javascript
// ── Mark conflicts ────────────────────────────────────────────────────────────

function markConflicts() {
  if (!scanResult || !scanResult.conflicts.length) {
    setStatus('Run Scan first.', 'error');
    return;
  }

  setStatus('Adding comments to document...');

  Word.run(function (ctx) {
    var searches = [];

    scanResult.conflicts.forEach(function (conflict) {
      if (conflict.type === 'number_reuse') {
        conflict.phrases.forEach(function (phrase) {
          var term = phrase + ' ' + conflict.number;
          var otherPhrases = conflict.phrases.filter(function (p) { return p !== phrase; });
          searches.push({
            term: term,
            comment: 'CONFLICT: number ' + conflict.number + ' also used for "' + otherPhrases.join('", "') + '"'
          });
        });
      } else if (conflict.type === 'phrase_reuse') {
        conflict.numbers.forEach(function (num) {
          var term = conflict.phrase + ' ' + num;
          var otherNums = conflict.numbers.filter(function (n) { return n !== num; });
          searches.push({
            term: term,
            comment: 'CONFLICT: "' + conflict.phrase + '" also numbered as ' + otherNums.join(', ')
          });
        });
      }
    });

    // Chain all searches sequentially (Office.js search returns lazy RangeCollection)
    var promise = Promise.resolve();
    var totalMarked = 0;

    searches.forEach(function (s) {
      promise = promise.then(function () {
        return Word.run(function (innerCtx) {
          var ranges = innerCtx.document.body.search(s.term, { matchCase: false });
          ranges.load('items');
          return innerCtx.sync().then(function () {
            ranges.items.forEach(function (range) {
              range.insertComment(s.comment);
              totalMarked++;
            });
            return innerCtx.sync();
          });
        });
      });
    });

    return promise.then(function () {
      setStatus('Added comments to ' + totalMarked + ' location(s). Review in the Comments pane.', 'success');
    });
  }).catch(function (err) {
    setStatus('Error marking conflicts: ' + err.message, 'error');
  });
}
```

```javascript
// ── Generate reference table ──────────────────────────────────────────────────

function generateTable() {
  if (!scanResult) {
    setStatus('Run Scan first.', 'error');
    return;
  }

  var tableData = scanResult.referenceTable;
  if (tableData.length === 0) {
    setStatus('No references found to tabulate.', 'error');
    return;
  }

  setStatus('Inserting reference table...');

  Word.run(function (ctx) {
    var body = ctx.document.body;

    // Insert section heading
    var heading = body.insertParagraph('Reference Numerals', Word.InsertLocation.end);
    heading.styleBuiltIn = Word.Style.heading2;

    // Build 2D values array: header row + data rows
    var values = [['Reference Number', 'Component Name']];
    tableData.forEach(function (row) {
      values.push([row.number, row.phrase]);
    });

    // Insert table
    var table = body.insertTable(values.length, 2, Word.InsertLocation.end, values);
    table.style = 'Table Grid';

    // Bold header row
    var headerRow = table.rows.getFirst();
    headerRow.font.bold = true;

    // Set column widths (DXA: 1 inch = 1440 units; content ~6.5" = 9360)
    var col0 = table.columns.getFirst();
    col0.getHeader(); // not needed, just ensure col exists
    // Column widths via preferred width on cells in header row
    headerRow.cells.getFirst().width = 2160;    // 1.5"
    headerRow.cells.getLast().width = 7200;     // 5"

    return ctx.sync().then(function () {
      setStatus('Reference table inserted at end of document (' + tableData.length + ' entries).', 'success');
    });
  }).catch(function (err) {
    setStatus('Error inserting table: ' + err.message, 'error');
  });
}
```

```markdown
# Patent Reference Checker — Installation Guide

## What this is

A Microsoft Word add-in that:
1. Scans patent specification text for reference number conflicts (e.g., number 102 used for both "interface" and "module")
2. Marks conflicts with Word comments
3. Generates a sorted reference numerals table

---

## Requirements

- Microsoft Word 2016 or later (Windows or Mac), or Word on Microsoft 365
- Python 3 (already installed on most machines — to check: open Command Prompt and type `python --version`)

---

## One-time setup (do this once per machine)

### Step 1 — Generate icons

Open Command Prompt in the `ms_word_addon` folder, run:
```

python generate-icons.py

Created assets/icon-32.png
Created assets/icon-80.png

```
1. Open Word.
2. Go to **File → Options → Trust Center → Trust Center Settings → Trusted Add-in Catalogs**.
3. In the "Catalog Url" box, type the full path to the `ms_word_addon` folder, e.g.:
   `C:\Users\YourName\Documents\ms_word_addon`
4. Click **Add catalog**.
5. Check **Show in Menu** for the catalog you just added.
6. Click **OK** twice and restart Word.

> **Tip for shared use:** If everyone maps the same network drive (e.g. `\\server\shared\ms_word_addon`), only one person needs to run the server — everyone else just sideloads the add-in.
```

python serve.py

Keep this terminal open while using the add-in in Word.

```
**Leave this terminal open** as long as you are using the add-in.

## Installing the add-in in Word

1. Open a patent specification document.
2. Go to **Insert → My Add-ins**.
3. Click the **SHARED FOLDER** tab.
4. Select **Patent Reference Checker** → click **Add**.
5. A button "Check References" appears in the **Home** tab ribbon.

---

## Using the add-in

1. Click **Check References** in the Home ribbon → the task pane opens on the right.
2. Click **Scan Document** → the pane shows:
   - Conflicts (red badges) — reference number inconsistencies
   - Reference Dictionary — all detected component-number pairs, sorted
3. Click **Mark Conflicts in Doc** → adds a Word comment to each conflict location. Review in the Comments pane (Review → Show Comments).
4. Click **Insert Reference Table** → inserts a two-column "Reference Numerals" table at the end of the document.
```
